# App-Tasker - Frontend

## About
This is the frontend client for the App-Tasker website. It is built using React to provide a modern, responsive user interface for managing tasks, handling user authentication, and interacting with the backend API.

## Setup and Installation

### 1. Clone the Repository & Setup Git
```bash
git clone https://github.com/aryan19022004/app-tasker.git
cd frontend  # or your frontend directory name
# Or if you are initializing an existing directory:
git init
git remote add origin https://github.com/aryan19022004/app-tasker.git
```

### 2. Install dependencies
Ensure you have Node.js installed, then run:
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```

This will start the Vite development server and you can view the application in your browser (typically at `http://localhost:5173`).
